Abstract
Automatic music recommendation has become an increasingly relevant problem in recent years, since a lot of music is now sold and consumed digitally. An essential step towards acheiving the solution to this problem is to teach computer about how to classify the music into various genres so that it can recommend similar kind of music with ease. Audio feature extraction helps us understand the properties of the audio signal and in turn helps to classify music.
1. Introduction
This is an audio signal processing project wherein we attempt to classify different music into specific genres using feature extraction followed by training classifiers using efficient machine learning algoritms. .
